# IEEE-SJCE Website v2 🚀

The new and improved version of **IEEE-SJCE's** official website — rebuilt from the ground up with the latest technologies, modern design principles, and a focus on performance and developer experience.

---

## ✨ What's New

- **Complete Redesign** — A fresh, neo-brutalist design language with bold visuals, smooth animations, and a premium feel.
- **Modern Tech Stack** — Built with the latest versions of React 18, Vite 6, Tailwind CSS 4, and more.
- **Enhanced Animations** — Powered by Framer Motion and GSAP for fluid, dynamic interactions.
- **Improved Performance** — Lightning-fast builds and hot module replacement with Vite.
- **Better Developer Experience** — Clean component architecture and modular codebase for easier contributions.

## 🚀 Getting Started

### Prerequisites

- **Node.js** — v18 or higher
- **npm** — v9 or higher (comes with Node.js)

### Installation

1. **Clone the repository**

   ```bash
   git clone https://github.com/IEEE-SJCE/ieee_v2.git
   cd ieee_v2
   ```

2. **Install dependencies**

   ```bash
   npm install
   ```

3. **Start the development server**

   ```bash
   npm run dev
   ```

4. Open [http://localhost:5173](http://localhost:5173) in your browser.
